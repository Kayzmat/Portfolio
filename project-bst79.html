AP1 - BTS SIO SLAM

Site WordPress
bst79

Réalisation d'un site WordPress pour l'entreprise réelle bst79. Projet réalisé en groupe de 4, avec intégration, configuration et optimisation du site existant.

WordPress PHP HTML CSS Responsive
Rôle Développeur web
Durée 28/11/2024 – 16/01/2025
Contexte AP1 - BTS SIO SLAM
Équipe Groupe de 4

Technologies

WordPress PHP HTML & CSS SEO Responsive

Objectifs

  • Créer ou refondre le site WordPress existant de bst79
  • Adapter le contenu à l'identité de l'entreprise
  • Rendre le site responsive et accessible
  • Optimiser les performances et le référencement

Détails du projet

Mise en ligne d'un site WordPress fonctionnel pour bst79, avec configuration du thème, adaptation du contenu, installation des plugins essentiels et mise en place d'un design cohérent.

Réalisations principales

  • Choix et personnalisation d'un thème WordPress adapté
  • Intégration de pages de présentation, contact et services
  • Configuration de plugins SEO, sécurité et sauvegarde
  • Optimisation pour mobile et compatibilité multi-navigateurs
  • Tests de navigation et validation finale avant mise en ligne

Ce que j'ai appris

🧩 Travail en équipe

Coordination et répartition des tâches entre 4 membres pour livrer le site ensemble.

⚙️ Configuration WordPress

Installation de thèmes et plugins, gestion des mises à jour et sécurisation du back-office.

📱 Adaptation responsive

Optimisation de l'affichage sur mobile et tablette pour une expérience utilisateur uniforme.

🔎 Référencement

Amélioration du SEO grâce à une structure de pages claire et un contenu optimisé.
