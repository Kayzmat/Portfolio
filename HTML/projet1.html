Projet 1 - Windows/Linux

Description du Projet


Ce projet consistait a créer une domaine sur windows 2019 avec un certain nombre de compte utilisateur et connecté à celui-ci 2 postes clients windows ainsi qu'un serveur web sur linux (debian). Ces postes windows devaient posséder un antivirus aisni que LibreOffice. Sur le serveur web on devait créer une page web et faire en sorte qu'on puisse y accéder à partir des clients windows.

Le projet est divisé en 4 parties

  • Première partie : création du domaine "humanlab.local" et des utilisateurs

  • Deuxième partie : Connexion au domaine des clients windows et installations des logiciels.

  • Troisième partie : Création et connexion au domaine de le poste debian.

  • Quatrième partie : Création de la page web sur le poste debian.

Défis

  • Connecté tout les postes au domaine.
  • Apprendre à utiliser un poste sur debian.
  • Créer une page web sur debian.

Contraintes appliquées

  • Carrousel d'images pour les produits.

  • Map dans le footer pour montrer l'emplacement de l'entreprise

  • Page mentions légales

  • Page d'erreur : la page où les utilisateurs vont être redirigés si l'url est mauvaise
